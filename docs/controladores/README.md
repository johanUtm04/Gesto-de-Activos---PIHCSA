# Resumen de controladores

Esta sección documenta los controladores principales del sistema.
Cada controlador es responsable de gestionar las solicitudes HTTP, coordinar la lógica de negocio y devolver las respuestas o vistas adecuadas.

Los controladores siguen las convenciones de Laravel y se agrupan por módulos funcionales para mejorar la mantenibilidad y la escalabilidad.

# Controllers Overview

This section documents the main controllers of the system.
Each controller is responsible for handling HTTP requests, coordinating
business logic, and returning the appropriate responses or views.

Controllers follow Laravel conventions and are grouped by functional modules
to improve maintainability and scalability.
